# pontia-git-practica

Práctica del módulo de **Versionado de Código** del Máster en IA, Cloud y DevOps de Pontia.tech.

En este repositorio se trabaja:

- Inicialización de un repositorio Git y primer commit.
- Protección de la rama `main` mediante reglas de GitHub.
- Uso de `.gitignore` para excluir artefactos del control de versiones.
- Flujo de ramas `feature/*` y Pull Requests.
- Resolución de conflictos de merge.
- Automatización con GitHub Actions (`workflow_dispatch`).
